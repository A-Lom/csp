Welcome to the Computer Science Puzzle

Computer Science Puzzle is a puzzle based website where you the player must find the password to the next page. Each page contains it's own puzzle which becomes harder as you continue.

How do you gain access to the puzzles

Each level can be access by adding /cspX to the end of the site url where X is the level number. For example level 1 is 10.8.26.135/csp1.
Each level has the password to the next level hidden within it, whether as plain text or in some other style.

The Username for each level is the level name.
Each password will always look like this when found. flag:1234567890(The password for the site is the part after flag:)

Password: csp1
URL: http://10.8.26.135/csp1
